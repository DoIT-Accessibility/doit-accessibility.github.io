ARIA Menu Button

A menu button is a button that opens a menu.

Example

  • Menu Item 1
  • Menu Item 2
  • Menu Item 3

Keyboard Interaction

  • Tab: Moves focus into and out of the widget. If focus was in the menu, closes the menu.
  • Enter or Space: If focus is on the button, opens the menu and places focus on the first menu item. If focus is on a menu item, activates the item and closes the menu.
  • Down Arrow or Up Arrow: If focus is in the menu, moves focus to the next or previous item.
  • Escape: If focus is in the menu, closes the menu and returns focus to the button.
