Keyboard Interaction

  • Tab: Moves focus in to and out of the editable element.
  • When focus is in the editable element, printable keys will enter characters into the element and navigation keys will move the cursor within the element.
  • If a screen reader activates its forms/focus mode, pressing Esc will deactivate that mode and allow use of Up/Down Arrow to move out of the element.

Roles, States, and Properties

  • The editable element has role textbox.
  • If the editable element is multiline, it has aria-multiline set to true.
  • The editable element has aria-labelledby or aria-label referring to or providing a label.
  • The editable element does not have aria-readonly set to true.
