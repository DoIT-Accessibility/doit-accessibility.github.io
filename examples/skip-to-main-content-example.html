Skip to Main Content Example

About This Example

The following example demonstrates how use a "Skip to Main Content" link to meet Success Criterion 2.4.1 Bypass Blocks using Technique G1.

There are other techniques, such as H69: Providing heading elements and ARIA11: Using ARIA landmarks, that satisfy 2.4.1, but providing a skip link is currently the only technique that works for keyboard-users who do not use screen readers or other assistive technologies.
