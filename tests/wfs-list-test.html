Note: These examples use several diffent methods of providing hidden, readable text (aria-label and sr-only) and associating labels (aria-label and aria-labelledby), which are noted in comments in the code. For actual use, you'd pick one method for each and use it consistently.

Ideally, the list or table would provide adequate context for the button in each list item/table row, but neither JAWS nor NVDA read the list item number if it is not displayed (e.g., because of list-style-type:none or display:flex). Because of this, the examples below use aria-label or sr-only text to explcitly identify the 'exception number'.
