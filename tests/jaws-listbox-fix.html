JAWS Listbox Fix

In Chrome or Edge, JAWS may incorrectly focus on and click the wrong button after making a selection from a listbox (select element with size greater than 1). For more details, see JAWS Listbox Test. Until JAWS is fixed, the bug can be avoided by using script to toggle the multiple attribute on the select element.

To test this fix:

  1. Tab to the Listbox below,
  2. Press down arrow to select an item,
  3. Tab to the Submit button,
  4. Press Enter or Space to click the Submit button,
  5. Confirm that the Submit button was clicked.

Script


// Apply to all select elements with size attribute and without multiple attribute (could use a classname instead)
document.querySelectorAll("select[size]:not([multiple])").forEach(function(listbox) { 
	// Double-check that size > 1 and multiple attribute was not set, since this fix will prevent multiple selections
	if (listbox.size > 1 && !listbox.multiple) { 
		// Set multiple attribute to avoid JAWS bug
		listbox.multiple = true; 
		// Remove multiple attribute when listbox is focused to prevent multiple selections
		listbox.addEventListener("focus", function(event) {
			listbox.multiple = false; 
		});
		// Reset multiple attribute when listbox loses focus
		listbox.addEventListener("blur", function(event) {
			listbox.multiple = true; 
		});
		// For browsers that don't fire the focus event (e.g., Mobile Safari), use the change event to check and clear multiple selections
		listbox.addEventListener("change", function(event) { 
			if (listbox.selectedOptions.length > 1) { 
				listbox.multiple = false;
			}
		});
	}
});
