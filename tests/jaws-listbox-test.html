JAWS Listbox Bug

The following example is an HTML select element that has a size attribute greater than 1 so that it is displayed as a listbox. In Chrome or Edge, JAWS incorrectly clicks the wrong button after making a selection from the listbox. NVDA does not have this problem, and JAWS does not have it with Firefox.

To replicate the error:

  1. Tab to the Listbox below (may need to Tab and then Shift + Tab),
  2. Press down arrow to select an item,
  3. Tab to the Submit button,
  4. Press Enter or Space to click the Submit button,
  5. Note that the Clear button is clicked instead of Submit.
