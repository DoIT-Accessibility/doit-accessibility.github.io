Image Button Test

An update to Google Chrome and Microsoft Edge (possibly version 131) casued these browsers to stop recognizing alt text on image buttons that have type="IMAGE" in all caps.

Test 1: type="image"

Read as "Test 1 button" as expected.

Test 2: type="IMAGE"

Read as "button" (NVDA, Narrator) or "Unlabeled 0 button" (JAWS); Should be read as "Test 2 button".
